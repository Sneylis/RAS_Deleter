# RAS Remover v2.0.0

Утилита для полного удаления Remote Access Software (RAS) с Windows-хостов с помощью встроенного MFT сканирования.

## Описание

**RAS Remover** — это Rust-приложение для удаления популярных инструментов удалённого доступа (RAS) с Windows-систем. Программа использует встроенное сканирование Master File Table (MFT) NTFS для эффективного поиска файлов на все NTFS-томах без внешних зависимостей.

Основной исполняемый файл: **ras_del.exe**

## Ключевые возможности

✓ **MFT сканирование** — встроенное сканирование NTFS Master File Table без внешних зависимостей
✓ **Многоуровневое обнаружение** — реестр (HKLM/HKCU) + процессы + MFT файловая система
✓ **Завершение процессов** — graceful stop → force kill с помощью taskkill
✓ **Встроенный деинсталлятор** — прямой вызов exe без ошибок парсинга
✓ **Удаление служб Windows** — через `sc delete` с проверкой статуса
✓ **Очистка файлов** — двухуровневая стратегия (Rust API → fallback на `cmd /c rd /s /q`)
✓ **Очистка реестра** — удаление из HKLM и HKCU Uninstall ветвей
✓ **Очистка файрвола** — удаление правил через `netsh advfirewall`
✓ **Silent режим** — скрытие консольного окна и подпроцессов
✓ **Dry-run режим** — проверка без изменений
✓ **Целевой инструмент** — удаление только конкретного RAS
✓ **Детальное логирование** — все операции записываются в лог-файл

## Использование

### Базовые команды

```cmd
REM Удалить все обнаруженные RAS (требуются права администратора)
ras_del.exe

REM Проверить что будет удалено (dry-run режим — без изменений)
ras_del.exe --dry-run

REM Скрыть консольное окно (silent режим)
ras_del.exe --silent

REM Удалить только конкретный инструмент
ras_del.exe --tool AnyDesk
ras_del.exe --tool TeamViewer
ras_del.exe --tool RustDesk
ras_del.exe --tool "Assistant FS"
```

#### Расширённые опции

```cmd
REM Сохранить логи в пользовательскую папку
ras_del.exe --log-dir C:\MyLogs

REM Комбинировать несколько флагов
ras_del.exe --dry-run --log-dir C:\Logs --silent
ras_del.exe --tool "Chrome Remote Desktop" --dry-run

REM Удалить с логами в AppData
ras_del.exe --silent --log-dir "%LOCALAPPDATA%\RAS_Remover\Logs"
```

## Флаги программы

| Флаг | Значение | Описание |
|------|----------|---------|
| `--dry-run` | — | Режим проверки без реальных изменений. Показывает что будет удалено. |
| `--silent` | — | Скрывает консольное окно и окна подпроцессов (taskkill, sc, netsh). |
| `--tool <NAME>` | AnyDesk, TeamViewer и т.д. | Удаляет только один конкретный RAS. Без флага удаляются все найденные. |
| `--log-dir <PATH>` | C:\Logs | Сохраняет логи в указанную папку. По умолчанию использует стандартные пути. |

### Примеры комбинаций

```cmd
REM Только проверка, никаких изменений
ras_del.exe --dry-run

REM Удалить только TeamViewer без консоли
ras_del.exe --tool TeamViewer --silent

REM Проверить что найдётся для конкретного RAS
ras_del.exe --tool AnyDesk --dry-run --log-dir D:\Logs

REM Полное удаление с логами в системную папку
ras_del.exe --log-dir "C:\ProgramData\Logs\RAS_Remover"
```

## Как это работает

### Общая схема

```
┌─────────────────────────────────────────────────────┐
│ 1. Проверка прав администратора                     │
│ 2. Инициализация логирования (в стандартную папку) │
│ 3. ФАЗА 1: СКАНИРОВАНИЕ (обнаружение RAS)          │
│ 4. ФАЗА 2: УДАЛЕНИЕ (6 этапов)                     │
│ 5. ФАЗА 3: ВЕРИФИКАЦИЯ (повторное сканирование)    │
│ 6. Финальный отчёт и код выхода                    │
└─────────────────────────────────────────────────────┘
```

### ФАЗА 1: Сканирование

Программа ищет установленные RAS по **четырём независимым источникам**:

#### 1.1 Реестр Windows (HKLM + HKCU)
- **HKLM:** `Software\Microsoft\Windows\CurrentVersion\Uninstall` — все пользователи
- **HKCU:** `Software\Microsoft\Windows\CurrentVersion\Uninstall` — текущий пользователь
- Извлекает: `InstallLocation`, `UninstallString`, `DisplayName`
- Находит даже скрытые установки

#### 1.2 Активные процессы
- Сравнивает имена процессов с таблицей RAS сигнатур
- Получает PID и полный путь к exe-файлу
- Помогает найти работающие экземпляры

#### 1.3 MFT сканирование (Master File Table)
- Сканирует встроенную NTFS таблицу на всех томах
- Ищет исполняемые файлы по именам (anydesk.exe, teamviewer.exe и т.д.)
- Не требует внешних инструментов
- Находит даже переносимые версии и нестандартные пути

#### 1.4 Стандартные пути установки
- `C:\Program Files\`
- `C:\Program Files (x86)\`
- `C:\ProgramData\`
- `%APPDATA%\` и `%LOCALAPPDATA%\`

**Результат:** Объединённый список с дедупликацией (case-insensitive)

### ФАЗА 2: Удаление (6 этапов)

#### Шаг 1: Завершение процессов
- Graceful stop (2 сек ожидания)
- Force kill `taskkill /F /IM <process_name>` если нужно

#### Шаг 2: Запуск деинсталлятора
- Парсит `UninstallString` из реестра
- Прямой вызов exe без cmd.exe (избегает ошибок парсинга)
- Ждёт корректного завершения

#### Шаг 3: Удаление служб Windows
- `sc delete <service_name>`
- Удаляет все связанные сервисы

#### Шаг 4: Удаление файлов и папок
- Двухуровневый подход:
  1. Встроенный Rust `fs::remove_dir_all()`
  2. Fallback на `cmd /c rd /s /q` при блокировке файлов

#### Шаг 5: Очистка реестра
- Удаляет ключи из HKLM и HKCU Uninstall веток
- Программа исчезает из "Программы и компоненты"

#### Шаг 6: Очистка файрвола
- `netsh advfirewall firewall delete rule name="..."`
- Закрывает все порты RAS инструмента

### ФАЗА 3: Верификация

После удаления программа **повторно сканирует** и проверяет:
- Процессы (должны быть удалены)
- Реестр (ключи должны отсутствовать)
- Файловая система (папки должны быть пусты)

**Результат:**
- ✓ **OK** — инструмент полностью удален
- ✗ **FAIL** — остались следы (обычно требует перезагрузки)

## Часто задаваемые вопросы (FAQ)

**Q: Может ли программа удалить RAS, если он запущен?**
A: Да. Программа сначала завершает процесс gracefully, затем force kill, затем удаляет остальное.

**Q: Требуются ли права администратора?**
A: Да, обязательно. Программа автоматически проверяет права при запуске.

**Q: Что делать если после удаления остались следы?**
A:
1. Проверьте логи в указанной папке
2. Перезагрузитесь (служба Windows может требовать перезагрузку)
3. Запустите программу снова
4. Если не помогает — проверьте права админа и занятые файлы


**Q: Может ли программа восстановить удаленное?**
A: Нет. После удаления требуется переустановка.

**Q: Работает ли на Windows 7?**
A: Нет, требуется Windows 10+. Старые версии имеют другой реестр.

**Q: Какой размер логов?**
A: Обычно 50-200 KB за одно выполнение.

**Q: Может ли программа удалить RAS если реестр повреждён?**
A: Да. MFT сканирование найдёт файлы даже если реестр неполный.
